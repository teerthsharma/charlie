All source code is available at `github.com/teerthsharma/charlie`.

## 2. Persistent Homology as Cosmological State Variable

### 2.1 Topological State

A **topological state** at simulation time *t* is a 5-tuple:

```
state(t) = (phase, B, ψ, barcode, S)
```

where:
- **phase** ∈ {VACUUM, H0_ONLY, H0H1, H0H1H2, GOD_FIXED} is the current cosmological phase
- **B** = (b₀, b₁, b₂) ∈ ℕ³ is the Betti signature — counts of H₀, H₁, H₂ persistent features
- **ψ** ∈ ℂ^(d^N) is the N-body Hilbert space vector of dimension *d*^*N* (tensor product ⊗ᵢ ψᵢ)
- **barcode** is the full set of (birth, death) persistence pairs for H₀, H₁, H₂
- **S** ∈ ℝ is the configurational entropy S = log(1 + total_features)

The Betti numbers are not parameters — they are **computed observables** from point-cloud data via persistent homology (ripser).

### 2.2 Persistent Homology

Given a point cloud {x_i} ⊂ ℝ^n, the **Vietoris–Rips filtration** builds nested simplicial complexes at scale ε:

```
K(ε₁) ⊆ K(ε₂) ⊆ ... ⊆ K(ε_max)
```

A *p*-dimensional homology group H_p appears at birth scale ε_b and disappears at death scale ε_d.  The pair (ε_b, ε_d) is a **persistence pair**.  The collection of all pairs across all dimensions is the **barcode**.  The count of H_p pairs with finite death is b_p — the **Betti number** b_p.

In the cosmological interpretation:
- **H₀** (b₀): connected components — matter overdense regions, galaxy filament junctions
- **H₁** (b₁): loops — quantum flux tubes, cosmic string analogues
- **H₂** (b₂): voids — gravitational cavities, dark energy regions

### 2.3 Phase Transition as Topological Bifurcation

The cosmological phase transitions are topological bifurcations — qualitative changes in the homology groups:

| Transition | Trigger | H change |
|---|---|---|
| ∅ → H₀_ONLY | First vacuum fluctuation (POVM E₀ outcome) | b₀: 0 → 1 |
| H₀_ONLY → H₀H₁ | Loop emergence (POVM E₁ outcome) | b₁: 0 → b₁ ≥ 1 |
| H₀H₁ → H₀H1H2 | Void formation | b₂: 0 → b₂ ≥ 1 |
| → GOD_FIXED | Banach fixed-point convergence (ε < 10⁻¹⁵) | All b_p stable |

## 3. The God Tensor — FARADAY Engine

### 3.1 Banach Fixed-Point for E↔H Coupling

The **God Tensor** G: ℂ^d → ℂ^d is defined as:

```
G(ψ) = T · ψ
```

where T ∈ ℂ^(d×d) is the electric-to-magnetic coupling matrix learned from observed (E, H) signature pairs via:

```
T = argmin_{T'} Σ_i ||T' · e_i - h_i||²
```

with (e_i, h_i) ∈ ℂ^d as electric/magnetic field signatures.

The God Tensor has a unique **fixed point** x* satisfying:

```
x* = T · x*
```

This is the **spectral fixed point** of the operator T.  By the Banach fixed-point theorem, iteration x_{n+1} = T · x_n converges to x* at machine epsilon (ε ≈ 1.8 × 10⁻¹⁶) if ||T|| < 1.  When T is learned from physical field data, it encodes the actual E↔H coupling of the vacuum.

### 3.2 Power Iteration

The fixed point is computed via power iteration:

```python
def power_iteration(self, psi0: np.ndarray | None = None, iters: int = 10000, tol: float = 1e-15) -> tuple[np.ndarray, float]:
    if psi0 is None:
        psi0 = self._random_unit_vector(self.latent_dim)
    psi = psi0 / (np.linalg.norm(psi0) + 1e-15)
    residual = float("inf")
    for _ in range(iters):
        psi_new = self.apply(psi)
        psi_norm = np.linalg.norm(psi_new)
        if psi_norm > 1e-15:
            psi_new = psi_new / psi_norm
        psi_new = np.nan_to_num(psi_new, nan=psi0[0] if len(psi0) else 1.0)
        if np.any(np.isnan(psi_new)):
            break
        residual = float(np.linalg.norm(psi_new - psi))
        psi = psi_new
        if residual < tol:
            break
    self.x_star = psi
    return psi, residual
```

The residual r_k = ||ψ_{k+1} - ψ_k|| is the **Banach residual** — when r_k < 10⁻¹⁵, the system has converged to the fixed point at machine precision.

### 3.3 God Score

The **god_score** measures how close a state ψ is to the fixed point:

```
god_score(ψ) = |⟨ψ|x*⟩|² = |ψ† · x*|²
```

This is the squared overlap (fidelity) with the cosmological attractor.  States near the fixed point have god_score → 1.

## 4. N-Body Quantum States — HAMLITON Engine

### 4.1 Tensor Product Hilbert Space

An N-body quantum state lives in the tensor product Hilbert space:

```
𝓗_total = 𝓗_1 ⊗ 𝓗_2 ⊗ ... ⊗ 𝓗_N
dim(𝓗_total) = d^N
```

where each body *i* has local dimension *d* (the latent dimension).  The full state vector |ψ⟩ ∈ ℂ^(d^N) is:

```python
def tensor_product(self, signatures: list[np.ndarray]) -> np.ndarray:
    result = signatures[0].flatten()
    for sig in signatures[1:]:
        result = np.kron(result, sig.flatten())
    result = result / (np.linalg.norm(result) + 1e-15)
    return result
```

For d=16, N=2: dim = 256.  For d=8, N=4: dim = 4096.  The state is normalized: |||ψ⟩|| = 1.

### 4.2 Learned Coupling Hamiltonian

The coupling Hamiltonian H ∈ ℂ^(d×d) is learned from paired signature observations {(ψ_i, ψ_j)} via:

```python
def learn_H(self, coupled_signatures: np.ndarray) -> None:
    n_bodies, d = coupled_signatures.shape
    if d != self.latent_dim * self.n_bodies:
        raise ValueError(f"Expected {self.latent_dim * self.n_bodies} cols, got {d}")
    self.H = np.zeros((self.latent_dim, self.latent_dim), dtype=np.float64)
    for coupled in coupled_signatures:
        signatures = np.split(coupled, self.n_bodies)
        for sig in signatures:
            self.H += np.outer(sig, sig) / len(coupled_signatures)
    self.H = self.H / (np.linalg.norm(self.H) + 1e-15)
```

H is an **entanglement-generating operator**: applying H ⊗ H via einsum('ik,jl,kl->ij', H, H, ψ) creates quantum correlations between bodies.

## 6. Stochastic Collapse — EPSILON-CLI Engine

### 6.1 POVM Framework

A **Positive Operator-Valued Measure** {E₀, E₁, E₂} on a 3-dimensional Hilbert space satisfies:

```
E_i ≥ 0    ∀i
Σᵢ E_i = I
tr(E_i) = p_i   (probability of outcome i)
```

The EPSILON-CLI engine implements:

```
E₀ = |00⟩⟨00|   (vacuum → no features)
E₁ = |01⟩⟨01|   (H₀_ONLY → connected components emerge)
E₂ = I − E₀ − E₁  (H₀H₁ → loops form; H₀H1H2 → voids form)
```

```python
class POVMMeasurement:
    def __init__(self) -> None:
        self.operators = [
            np.array([[1, 0, 0], [0, 0, 0], [0, 0, 0]], dtype=np.float64),  # E0
            np.array([[0, 0, 0], [0, 1, 0], [0, 0, 0]], dtype=np.float64),  # E1
            np.array([[0, 0, 0], [0, 0, 0], [0, 0, 1]], dtype=np.float64),  # E2
        ]

    def sample(self, state: np.ndarray, rng: np.random.Generator) -> int:
        probs = np.array([np.real(np.conj(state) @ E @ state) for E in self.operators])
        probs = np.maximum(probs, 0.0)
        probs /= probs.sum()
        return rng.choice(len(self.operators), p=probs)
```

### 6.2 Vacuum Fluctuation

The **StochasticBang** class generates vacuum fluctuations:

```python
def vacuum_fluctuation(self, rng: np.random.Generator) -> BettiSignature:
    noise = rng.standard_normal(3) * self.noise_amplitude
    b0 = max(0, int(round(noise[0])))
    b1 = max(0, int(round(noise[1])))
    b2 = max(0, int(round(noise[2])))
```

When the system is in VACUUM (b₀=b₁=b₂=0), the POVM samples from this distribution — the initial cosmological perturbation that breaks the empty set symmetry.

### 6.3 POVM Collapse

```python
def povm_collapse(
    self,
    betti: BettiSignature,
    noise_amplitude: float,
    rng: np.random.Generator,
) -> BettiSignature:
    state = np.array([betti.b0 + 1, betti.b1 + 1, betti.b2 + 1], dtype=np.float64)
    state = state / (np.linalg.norm(state) + 1e-15)
    outcome = self.povm.sample(state, rng)
    noise = rng.standard_normal(3) * noise_amplitude
    b0, b1, b2 = betti.b0, betti.b1, betti.b2
    if outcome == 0:
        b0 = max(0, b0 - 1)
    elif outcome == 1:
        b1 = max(0, b1 - 1)
    else:
        b2 = max(0, b2 - 1)
    return BettiSignature(b0=b0 + int(round(noise[0])), b1=b1 + int(round(noise[1])), b2=b2 + int(round(noise[2])))
```

The **POVM collapse** is the mechanism by which topological features appear or disappear as the simulation evolves.  The outcome is deterministic given the state vector, but the state itself evolves via God Tensor application and N-body coupling.

## 7. Phase Manager and Simulation Orchestration

### 7.1 Phase Detection

```python
def detect_phase(self, betti: BettiSignature) -> TopologyPhase:
    if betti.is_vacuum:
        return TopologyPhase.VACUUM
    elif betti.b2 > 0:
        return TopologyPhase.H0H1H2
    elif betti.b1 > 0:
        return TopologyPhase.H0H1
    elif betti.b0 > 0:
        return TopologyPhase.H0_ONLY
    return TopologyPhase.VACUUM
```

Phase detection uses the **actual computed Betti numbers** from ripser barcodes — not stochastic draws.  This is the key distinction from earlier toy models: the phase transition criterion is topological, not probabilistic.

### 7.2 Full Simulation Loop

The `BigBangSimulation.run()` method orchestrates all four engines:

```python
def run(self) -> dict[str, Any]:
    for step in range(self.max_steps):
        state = self.phase_manager.evolve_phase(state, self.rng)
        self.phase_manager.memory.store(state)
        if self._should_stop(state):
            break
    return self._build_result(state, step + 1)
```

The stopping criterion combines phase detection and God Tensor convergence:

```python
def _should_stop(self, state: TopologicalState) -> bool:
    if self.target_phase is not None and state.phase == self.target_phase:
        return True
    if state.residual is not None and state.residual < 1e-15:
        return True
    if state.phase == TopologyPhase.GOD_FIXED:
        return True
    return False
```

## 9. Open Questions and Future Work

1. **Initial field data:** Replace `rng.standard_normal` E/H signatures with measured physical field data (e.g., CMB polarization maps, LIGO strain data) to make the God Tensor a real predictive tool rather than a synthetic demonstration.

2. **Standard Model from Betti invariants:** Can the three generations of fermions, four force carriers, and the Higgs boson be classified as topological defects in the barcode structure?  This would connect persistent homology to the observed particle spectrum without introducing gauge groups by hand.

3. **Dimension selection:** Why does our universe have exactly 3 spatial dimensions?  In the topological picture, d=3 may emerge as the only dimension in which H₂ (voids) are stable under persistent homology perturbation — all other dimensions have different Betti number growth rates.

4. **Dark energy from barcode topology:** The cosmological constant Λ may correspond to the fraction of "infinite death" H₂ voids (unstable cavities that never collapse).  Quantifying this fraction from actual cosmological barcodes is an open problem.

5. **ripser + FAISS scaling:** Current implementation is single-threaded.  For cosmological simulations with >10⁶ point-cloud samples, distributed ripser computation and GPU-accelerated FAISS are required.
